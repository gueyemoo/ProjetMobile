La multiplication



I. Sens de la multiplication



Faire une multiplication équivaut à ajouter plusieurs fois le même nombre.
Exemple : 23 + 23 + 23 + 23 = 4 x 23 (4 fois  23) = 92

Attention : il faut apprendre et connaître les tables de multiplication pour bien multiplier.

II. La multiplication


On pose la multiplication en respectant les ordres. Les unités sous les unités, les dizaines sous les dizaines, les centaines sous les centaines, etc.
Le nombre placé en haut est multiplié par celui placé dessous.
Exemple 45 X 5 se pose :

     4 5 ----> multiplicande

X     5 ----> multiplicateur

  2 2 5 ----> produit



III. La multiplication : technique opératoire


Technique opératoire : plusieurs chiffres au multiplicateur

Multiplier 532 par 23 équivaut à multiplier 500 + 30 + 2 par 3 puis par 20.

Première étape: On multiplie 532 par le chiffre des unités : 3
On obtient alors le premier résultat intermédiaire : 1596

Deuxième étape : on multiplie 532 par le chiffres des dizaines : 2.
On place d’abord un 0, car cela équivaut à multiplier par 20.
(2 dizaines = 20 unités)
On a le deuxième résultat intermédiaire : 10640

Troisième étape : on ajoute les résultats intermédiaires.
Le résultat final est donc 12236
